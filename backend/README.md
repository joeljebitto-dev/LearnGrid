# Backend Services

The backend contains Django REST Framework service baselines for `SVC-001` through `SVC-010`.
Each service exposes `GET /health/` and follows the structure in
[../docs/BACKEND_ARCHITECTURE.md](../docs/BACKEND_ARCHITECTURE.md).

The repeated service skeletons are generated by [../tools/scaffold_backend_services.py](../tools/scaffold_backend_services.py).
